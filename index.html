Política de Privacidade

Este aplicativo utiliza a API oficial do WhatsApp Business (Meta).

Não coletamos, armazenamos ou compartilhamos dados pessoais dos usuários, exceto quando necessário para o funcionamento do serviço.

As mensagens trocadas são utilizadas exclusivamente para comunicação entre o usuário e a empresa.

Não vendemos, alugamos ou compartilhamos dados com terceiros.
